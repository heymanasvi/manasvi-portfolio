# Manasvi Gupta — Portfolio

Personal portfolio website built with React + Vite + Tailwind CSS.

## Tech Stack
- React 18
- Vite
- Tailwind CSS
- Font Awesome (icons)
- EmailJS (contact form)

## Run locally
```bash
npm install
npm run dev
```

## Deploy
Connected to Vercel. Push to `main` branch to auto-deploy.
